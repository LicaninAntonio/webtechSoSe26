# webtechSoSe26

Eine Webanwendung zur Verwaltung von Haushaltsfinanzen. Nutzer können Einnahmen und Ausgaben erfassen, 
in Kategorien einordnen, Budgets für diese Kategorien setzen und ihre finanzielle Entwicklung über verschiedene 
Zeiträume hinweg verfolgen. Ziel ist es, einen klaren Überblick über die eigene Haushaltskasse zu schaffen und 
das Budgetmanagement im Alltag zu erleichtern.
